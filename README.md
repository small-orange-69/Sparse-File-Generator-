# 稀疏文件创建者 - Sparse File Generator

一个支持多语言的现代化稀疏文件创建工具，可以创建看起来很大但实际上占用极少磁盘空间的文件。

## 🌟 功能特性

### 🌍 多语言支持
- **7种语言完整支持**：简体中文、English、日本語、한국어、Français、Deutsch、Español
- **实时语言切换**：无需重启程序即可切换界面语言
- **本地化界面**：所有界面元素、提示信息、错误消息都已完全本地化

### 🖥️ 现代化界面
- **直观的GUI界面**：简洁易用的图形用户界面
- **多框架布局**：清晰的输入区域和功能分区
- **状态栏显示**：实时显示程序状态信息
- **菜单系统集成**：文件、设置、帮助等完整菜单功能

### 🔧 核心功能
- **稀疏文件创建**：高效创建大文件，实际占用空间极小
- **多单位支持**：支持 B, KB, MB, GB, TB, PB, EB, ZB, YB 等单位
- **智能路径处理**：自动处理文件路径和扩展名
- **文件存在检查**：智能检测文件冲突并提供覆盖选项

### 💻 使用模式
- **图形界面模式**：双击运行或通过开始菜单启动
- **命令行模式**：支持批处理和脚本自动化

## 🚀 使用方法

### 图形界面模式
1. 双击运行程序或从开始菜单启动
2. 选择界面语言（右上角下拉菜单）
3. 输入文件大小（如：1GB, 500MB, 2TB）
4. 输入文件名
5. 选择保存位置
6. 点击"生成"按钮

### 命令行模式
```bash
python 稀疏文件.py 1GB 测试文件.bin D:\保存路径\
```

参数说明：
- 参数1：文件大小（支持 B, KB, MB, GB, TB, PB, EB, ZB, YB）
- 参数2：文件名
- 参数3：保存路径

## 💡 使用技巧

### 大文件创建
- 对于超大文件（如TB级别），建议使用命令行模式
- 确保目标磁盘有足够的可用空间用于文件系统元数据

### 性能优化
- 使用SSD存储可以提高文件创建速度
- 避免在系统盘创建过大的稀疏文件

### 安全建议
- 不要将重要数据存储在稀疏文件中
- 定期备份重要文件
- 使用可靠的存储介质

### 使用须知
- 稀疏文件并非真实数据，切勿用于存储重要信息
- 请在可靠、有足够空间的磁盘分区上操作
